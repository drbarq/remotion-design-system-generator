### 3.2 Invariant Component Specifications (AI-Styled)

The pipeline uses 5 locked, structurally rigid React components. The AI engine is strictly forbidden from modifying HTML/TSX layout trees; its sole capability is mapping extracted `tokens.json` values directly to the predefined CSS/Tailwind property hooks:

1. **`MediaFrame`**: Responsive wrapper for screenshots/images. Evaluates asset aspect ratio to apply 70%/90% screen height rules, rendering localized borders, corner radii (`spatial.radius`), and background plate separation.
2. **`KeyTermChip`**: A two-tier conceptual overlay taking up roughly one-third of the screen canvas. Features a prominent master term bound to `palette.accent` with a secondary, smaller typography layer for definitions.
3. **`TextOverlay`**: High-impact text callout blocks for highlight slides, quotes, or important notes. Completely decoupled from standard video subtitles.
4. **`TitleCard`**: Large-scale hero canvas for chapter breaks and initial script hooks, inheriting massive font scaling and core background tones.
5. **`LowerThird` / `StepBadge`**: Structural tracking components for contextual text callouts (links/handles) and sequential process indexing (Step 1, Step 2).
